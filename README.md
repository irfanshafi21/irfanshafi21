## Featured Work

### 🔹 Intelligent Candidate Discovery — *AI Recruitment Screening Tool*

Automates candidate shortlisting by scoring resumes against a job description — replacing hours of manual review with an instant, ranked shortlist.

- Bulk resume parsing across multiple candidates
- Configurable scoring by skills, experience, or project activity
- JD-driven relevance scoring beyond keyword matching
- Returns a ranked Top-10 shortlist for faster hiring decisions

**Stack:** Python · NLP (TF-IDF, Cosine Similarity) · Pandas · Scikit-learn
**Status:** In active development
[→ View Repository](https://github.com/irfanshafi21/intelligent-candidate-discovery)

<br>

### 🔹 EduTrack — *Academic Management Platform (Live)*

A role-based web application giving teachers real-time class oversight and students a clear view of their own academic progress.

- Role-based dashboards for teachers and students
- Automated attendance percentage & GPA calculation
- Weekly timetable views by department and semester
- Live announcements and visual performance analytics

**Stack:** HTML5 · JavaScript · Tailwind CSS · Chart.js
**Status:** Live in production
[→ View Repository](https://github.com/irfanshafi21/edutrack) · [→ Live Demo](https://irfanshafi21.github.io/edutrack/)
