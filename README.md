## Project — Intelligent Candidate Discovery

A tool that streamlines resume screening using machine learning.

**Workflow**

| Step | Action |
|------|--------|
| 01 | Upload multiple candidate resumes |
| 02 | Upload the job description |
| 03 | Select match criteria — skills, experience, or activity |
| 04 | System scores and ranks all candidates |
| 05 | Returns the top 10 best-fit candidates |

**Highlights**
- Bulk resume parsing
- Configurable matching weights (skill / experience / activity)
- JD-driven candidate scoring, not just keyword search
- Clean, ranked shortlist output

`Python` · `NLP` · `Machine Learning` · `Pandas`
